## Dataset Source
IMDb Non-Commercial Dataset:  
https://datasets.imdbws.com/

Files mainly used:
- title.basics.tsv.gz
- title.ratings.tsv.gz
- name.basics.tsv.gz
- title.crew.tsv.gz (optional for director mapping)

## Project Objective
To explore IMDb datasets and uncover hidden patterns in:  
genres, movie rating trends, audience preferences, director impacts, runtime correlation, etc.
